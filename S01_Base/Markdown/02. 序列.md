# Python序列（Sequence）
1. 这里先介绍以下 **容器**（Container）。

>容器是一种内置或自定义的数据结构，用于存储和组织其他对象。  
>* 容器和元素之间是包含关系：即可测试一个元素是否存在于该结构中（in 或 not in）。  
>* 容器可计算大小：可以使用 `len()` 函数获取其包含的元素数量。
>
>容器可以分为**序列**和**其他容器**。

2. 序列是 Python 中最基本的数据结构，序列中的每个值都有对应的位置值，称之为索引。序列中的元素可通过索引（位置偏移）进行访问。
**Python 有 6 个序列的内置类型，常用的有三种：`list`、`tuple`、`str`**。


3. 序列的核心特性

>* 有序：元素按照特定的顺序排列。  
>* 索引：每个元素都有一个唯一的编号索引，从 0 开始递增。也支持从末尾（-1）开始索引。  
>* 遍历：可以使用 `for...in` 循环来依次访问序列中的每个元素。  
>* 可切片：可以通过切片操作来获取序列的一个子集。  
>* 可计算长度：使用内置函数`len()` 可以获取序列中包含的元素个数。  
>（使用`len()`可以计算序列的长度，可以计算集合、字典中去重后的元素个数。）

4. 其它容器：`set`、`dict`、`frozenset`。

   * 非序列类型的容器同样可以进行迭代，序列和非序列均是可迭代对象（iterable）。
   * 也就是说，可以使用`tuple()`、`list()`、`set()`对其进行格式转换。
   * 另外，`str()`、`dict()`部分支持，`int()`、`float()`、`bool()`、`complex()`等不支持。
