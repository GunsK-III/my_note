网页三大技术要素
    CSS：        定义网页样式
    JavaScript： 定义用户和网页交互逻辑
    HTML：       定义网页结构和信息


简易的HTML样式：
<!DOCTYPE HTML>     # 声明文件类型是HTML
<html>              # 起始标签表示开始
    <head>          # 定义文档头部
        <title>Title</title>        # 展示在标签页选项卡上的标题
    </head>
    <boby>          # 表示文档的主体内容，同时是起始标签
        <h1>...</h1>        # 表示一级标题
        <p>...</p>         # 表示文本段落，这两个元素都是<body>元素的子元素
    </boby>         # 闭合标签
</html>             # 闭合标签表示结束
起始标签和闭合标签连同其中内容，表示一个HTML元素，
所有的其他元素都要放到这个元素之中


具体讲一下常用的标签类型：
标题标签：
<h1>一级标题</h1>
<h2>二级标题</h2>
...
文本段落标签：
<p>
