# pip —— Python包管理工具
>* pip 全称是Pip Installs Packages，使用了有意思的递归缩写
>* pip 工具默认随 Python 一起安装
>* 可以通过 `pip --version` 或 `python -m pip --version` 检查版本


### 常用 <code>pip</code> 命令
| 用法     | 命令                                                                                 |
|:-------|:-----------------------------------------------------------------------------------|
| 帮助     | <code>pip help</code>                                                              |
| 安装包    | <code>pip install package_name</code>                                              |
| 安装指定版本 | <code>pip install package_name==1.2.3</code>                                       |
| 升级包    | <code>pip install -U package_name</code>                                           |
| 卸载包    | <code>pip uninstall package_name</code>                                            |
| 查看包    | <code>pip list</code>                                                              |
| 查看可升级包 | <code>pip list --outdated</code>                                                   |
| 查看包详情  | <code>pip show package_name</code>                                                 |
| 导出依赖列表 | <code>pip freeze > requirements.txt</code>                                         |
| 安装依赖列表 | <code>pip install -r requirements.txt</code>                                       |
| 清华源    | <code>pip install -i https://pypi.tuna.tsinghua.edu.cn/simple/ package_name</code> |
